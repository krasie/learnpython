# learnpython

# 1、count_pic_daily.py
这是一个统计用户目录下面图片数量的程序，以日期为单位，区分图片和视频的数量。

# 2、dir_rename.py
提供一个修改前名字和修改后名字的列表，查找列表，寻找对应的文件夹，并修改名字，没有修改成功的记录到lost.txt文件里面。
